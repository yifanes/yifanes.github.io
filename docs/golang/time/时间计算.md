# 时间计算相关的片段代码

## 将字面量的数字转换为Duration

1.通过time.ParseDuration()

```go
time.ParseDuration("1h1m1s") //int64(Duration)
```

2.如果是数字直接转

```go
time.Duration(-120) * time.second //这里只是个类型转换而已
```


## 时间相加减

1.time.Now().Add()

> Add方法接受Duration参数,返回`time`类型

2.time.Sub()

> 接受time类型的参数,返回Duration类型的结果

## 时间格式化format

```go
now := time.Now().Format("2006-01-02 15:04:05") //返回strin
now = time.Now().Add(time.Hour * 1).Format("2006-01-02 15:04:05")//string
```
