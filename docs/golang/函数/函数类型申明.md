# 函数类型申明function types

```go
package main

import "fmt"

type Greeting func(name string) string

func (g Greeting) Say2(name string) string {
	return g(name)
}

func Say(greeting Greeting, name string) string  {
	return greeting(name)
}

func china(name string) string {
	return "hello, " + name
}

func main() {
	s1 := Say(china, "aaa")
	fmt.Println(s1)

	g := Greeting(china)
	s2 := g.Say2("xixi")
	fmt.Println(s2)
}
```

核心思想:

1. say方法可以接受func类型的变量
2. func类型的变量可以添加方法(如上例子中的say2)
