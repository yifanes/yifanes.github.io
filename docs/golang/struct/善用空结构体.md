# 善用空结构体



空结构体本身不占用系统内存,作为set来使用,利用ok特性可以解决好多问题

```go
set := make(map[int]struct{})
set[1] = struct{}{}
set[2] = struct{}{}

if _, ok := set[1]; ok {

}
```

用o(1)的复杂度,解决o(n)

将slice 转化为 map[string]struct{}



```go
//slice 求diff
func difference(a, b []string) []string {
    mb := make(map[string]struct{}, len(b))
    for _, x := range b {
        mb[x] = struct{}{}
    }
    var diff []string
    for _, x := range a {
        if _, found := mb[x]; !found {
            diff = append(diff, x)
        }
    }
    return diff
}
```
